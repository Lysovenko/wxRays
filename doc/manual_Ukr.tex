\section{Встановлення}
Для встановлення wxRays ваша система повинна містити:
\begin{itemize}
\item \href{http://python.org/}{Python}
\item \href{http://www.wxpython.org/}{wxPython}
\item \href{http://www.numpy.org/}{NumPy}
\item \href{http://www.scipy.org/}{SciPy}
\item \href{http://matplotlib.org/}{Mathplotlib}
\end{itemize}

Якщо назване ПЗ не встановлено на вашій системі - його потрібно встановити в зазначеному вище порядку. Для запуску програми достатньо розпакувати вміст архіву з дистрибутивом в зручну для вас директорію та запустити з допомогою Python файлу {\ttfamily \_\_main\_\_.py}. 

\section{Базові особливості програми}
При вимкнених додатках програма може завантажувати двоколонковий коментований текстовий файл {\ttfamily *.dat} та відображати його вміст на графіку. Дані, що відображаються на графіку можуть бути збережені як малюнок або як коментований {\ttfamily *.dat} файл.

Додатки можуть створювати додаткові коментарі до графіка - їх можна відобразити комбінацією клавіш Ctrl+D, або з допомогою відповідного пункту в меню. Коментарі зберігаються в файлах {\ttfamily *.dat} при збереженні даних з графіка.

Перемикання між різними графіками здійснюється таким чином: {\bfseries Меню~$\to$ Графік (Plot)~$\to$ Назва потрібного графіка}.

Іноді може виникнути потреба залишити лише частину експериментальних даних. Для цього потрібно виділити на графіку з ними ділянку, яку потрібно залишити (враховується лише абсциса) та вибрати {\bfseries Меню~$\to$ Дані (Data)~$\to$ Обрізати експ. дані (Crop exp. data)}.


\section{Налаштування додатків}
Налаштування додатків здійснюється через {\bfseries Меню~$\to$ Інструменти (Tools)~$\to$ Додатки\ldots (Addons\ldots)}. Якщо додаток може налаштовуватись, відмічений в списку (<<галочкою>>) та виділений "--- засвічується кнопка <<Налаштувати\ldots>>. До додатків, які можуть налаштовуватись належать: <<Alternative for gettext>> та <<NIST database browser>>. Перший потребує вказування файлу {\ttfamily *.po}, який містить переклад повідомлень програми, другий "--- місцезнаходження файлу {\ttfamily pdf2.dat}, який містить рентгено-дифракційну базу даних PDF-2 або її аналог.

\section{Опис стандартних додатків}

\subsection{Alternative for gettext}
Додаток являє собою альтернативу для GNU gettext "--- інструменту для інтернаціоналізації. Його варто використовувати в системах, де gettext не підтримується. Вибір файлу з перекладом здійснюється при налаштуванні додатка, а не автоматично в залежності від вибраної локалі, як у випадку з gettext. Вмикання цього додатку призводить до перезавантаження всіх інших додатків. Тому його бажано вмикати лише на початку або при завершенні роботи з програмою "--- коли немає ризику втрати даних. 

\subsection{Liquids and amorphous matherials data processor}
Додаток використовується для розрахунку загальних структурного фактора та парної кореляційної функції для рідких та аморфних матеріалів.

\subsection{NIST database browser}
Додаток призначений для пошуку та перегляду шаблонів в рентгенодифракційній базі даних PDF-2. Файл БД потрібно задати при налаштуванні додатку.

\subsection{Powder data processor}
Додаток призначений для обробки порошкових рентгенограм.

\subsubsection{Розробницькі примітки}
\begin{description}
\item [<<User refl>>] рефлекси, визначені користувачем. Являє собою об'єкт \texttt{PredefRefl}.
\item [<<Reflexes>>] рефлекси, вирахувані з рентгенограми~--- словник, в якому поле \texttt{<<shape>>} вказує на тип апроксимуючої кривої (\texttt{<<Gaus>>, <<Lorentz>>, <<P-Voit>>}), поле \texttt{<<items>>} являє собою список кортежів, які містять параметри описуючих функцій ($x_0$, $\sigma$, $h$) та стандартне відхилення знайденої функції від експериментальної кривої в межах рефлексу.
\item [<<Background>>] крива фону, яка є кортежем: $sin(\theta)$, $I$, функція фону, $\sigma^2$ кривої фону, коефіцієнти поліному.
\end{description}
\section{Встановлення сторонніх додатків}
Додаток складається з файлу {\ttfamily *.addon}, в якому міститься опис додатка, та одного або декількох файлів {\ttfamily *.py}.
Для встановлення сторонніх додатків потрібно їх помістити в
директорію, в якій розміщена програма або в директорію, де вона
зберігає свої налаштування. В POSIX-сумісних системах програма
зберігає свої налаштування в директорії \mbox{\ttfamily $\sim$/.wxRays} або \mbox{\ttfamily $\sim$/.config/wxRays} у всіх інших системах налаштування зберігаються в каталозі {\ttfamily wxRays} який розміщений в домашньому каталозі користувача.
