\part*{Вступ}
wxRays задумувалась як легкорозширювана програма зі зручним інтерфейсом для обробки рентгенодифракційних даних. Програма є відкритим кодом на мові Python.
% Відкритість коду дає можливість 
В якості графічного інтерфейсу wxRays використовує wxPython та Matplotlib. Вибір впав на wxPython, оскільки він містить широкий набір елементів інтерфейсу і добре документований. Для прискорення роботи з числовими масивами використовується бібліотека NumPy.
